# projectSiwoo

## 개요 : 부동산 전월세에 대한 비교자료 프로젝트

### 내용
- 🏠Home
    + 최근 한달 실거래 현황(최신순)
- 🔎전월세 검색
    + 구, 동, 전세/월세, 보증금, 임대면적 선택 후 조회
    + 계약일 기준 거래내역 표로 표현
- 📊전세예측
    + 구 선택 시, 전세 월평균 그래프
    + 월세 월평균 -> 보증금 월평균, 월세 월평균 그래프 표현
- 🤖챗봇
    + GPT-3 api를 이용한 chatbot
    + 서울시 공공데이터를 통한 날짜별 자치구 거래내역 조회 가능
    + 이외에 GPT-3를 통해 학습시켜 놓은 데이터를 바탕으로 AI 채팅
- 💬건의사항
    + 문의하기를 통해 사용자의 불만이나 궁금증을 접수
    + 검색 기능으로 건의사항을 검색
    + 자주 묻는 질문과 목록 탭
